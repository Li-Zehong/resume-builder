# 📝 简历生成器 Resume Builder

一个基于 React + TypeScript + Tailwind CSS 的在线简历编辑器，支持 Markdown 实时预览、自定义标签语法、PDF 导出，采用 CodeCV 风格排版。

---

## ✨ 功能特性

- 📝 **Markdown 实时编辑** — 右侧抽屉式编辑器，带语法高亮，所见即所得
- 🏷️ **丰富的自定义标签** — 支持彩色标签、四列 Flex 布局、公司 Logo、内联图标等扩展语法
- 📄 **导出 PDF** — 支持浏览器打印（推荐）和 html2canvas 两种方案，智能分页
- 💾 **自动保存** — 数据持久化到 localStorage，刷新不丢失
- 📂 **导入/导出 Markdown** — 支持 `.md` 文件的导入导出，方便多端迁移
- 🎯 **点击定位** — 点击预览区域的任意内容，编辑器自动跳转到对应行
- 📏 **行距调节** — 顶部滑块实时调整简历行间距
- 🌙 **暗色编辑界面** — 护眼的 GitHub Dark 风格编辑器

## 🚀 快速开始

### 环境要求

- **Node.js** >= 16
- **npm** >= 7（或 pnpm / yarn）

### 安装依赖

```bash
cd resume-builder
npm install
```

### 启动开发服务器

```bash
npm run dev
```

打开浏览器访问 **http://localhost:5173**

### 构建生产版本

```bash
npm run build
```

构建产物位于 `dist/` 目录，可直接部署到静态服务器。

## 📖 使用指南

### 1. 界面布局

| 区域 | 说明 |
|------|------|
| **顶部导航栏** | 包含 Logo、行距滑块、编辑/导入/导出/重置按钮、PDF 导出按钮 |
| **中间预览区** | 以 A4 纸张尺寸（794×1123px）实时渲染简历 |
| **右侧编辑器抽屉** | 点击「✏️ 编辑简历」按钮或右侧拉手打开，支持 Markdown 语法高亮 |
| **底部状态栏** | 显示自动保存状态 |

### 2. 编辑简历

点击顶部的 **「✏️ 编辑简历」** 按钮或右侧紫色拉手，打开编辑器抽屉。在编辑器中使用 Markdown 语法编写内容，左侧会实时更新预览。

编辑器顶部提供 **格式工具栏**，可快速插入：

| 按钮 | 功能 | 语法 |
|------|------|------|
| `H1` | 一级标题 | `# 标题` |
| `H2` | 二级标题 | `## 标题` |
| `B` | 加粗 | `**文本**` |
| `•` | 列表项 | `- 内容` |
| `红` | 红色标签 | `~~文本~~` |
| `紫` | 紫色标签 | `^^文本^^` |
| `绿` | 绿色标签 | `++文本++` |
| `!` | 高亮标签 | `!!文本!!` |
| `Flex` | 四列布局 | `@flex{A\|B\|C\|D}` |

### 3. 点击定位编辑

在预览区域**点击任意内容**（标题、段落、列表项、Flex 布局等），编辑器会自动打开并平滑滚动到对应行，同时高亮闪烁提示当前行位置。

## 🏷️ 自定义 Markdown 扩展语法

除了标准 Markdown 语法外，本工具支持以下扩展语法，专为简历排版设计：

### 头像

```markdown
@avatar{图片URL}
```

示例：

```markdown
@avatar{https://api.dicebear.com/7.x/avataaars/svg?seed=zhangsan&backgroundColor=b6e3f4}
```

效果：在简历右上角显示头像照片（90×115px）。

### 四列 Flex 布局

```markdown
@flex{第一列 | 第二列 | 第三列 | 第四列}
```

示例：

```markdown
@flex{**华南农业大学** ~~双一流~~ | **本科（学习委员）** | **人工智能专业** | **2022.09-2026.07**}
```

效果：四列水平排列，列宽根据内容**自动计算**。第一列靠左、中间两列居中、第四列靠右显示。

**折叠模式**：当中间两列为空时，自动折叠为两端对齐：

```markdown
@flex{**项目名称** `标签1` `标签2` | | **角色** | **时间**}
```

### 公司 Logo

```markdown
@company{公司名}
@company{公司名|自定义Logo URL}
```

示例：

```markdown
@flex{@company{美团} | **酒店旅行技术部** | **开发实习生** | **2025.05-2025.07**}
@flex{@company{oiloil.ai|https://example.com/logo.png} | **部门** | **职位** | **时间**}
```

**内置公司 Logo**：小红书、美团、阿里/阿里巴巴、腾讯、字节跳动/字节、百度、京东、华为、快手、拼多多、B站/哔哩哔哩。

### 内联图标

```markdown
@icon{图标名}
```

可用图标列表：

| 图标名 | 说明 | 图标名 | 说明 |
|--------|------|--------|------|
| `user` / `person` | 人物 | `phone` | 电话 |
| `email` / `mail` | 邮箱 | `location` / `map` | 定位 |
| `link` | 链接 | `github` | GitHub |
| `calendar` | 日历 | `male` / `female` | 性别 |
| `star` | 星星 | `trophy` | 奖杯 |
| `graduation` | 毕业帽 | `briefcase` | 公文包 |

### 彩色标签

| 语法 | 效果 | 样式说明 |
|------|------|----------|
| `` `文本` `` | 代码标签 | 灰底红字加粗 |
| `~~文本~~` | 红色标签 | 灰底红字加粗（与代码标签一致） |
| `^^文本^^` | 紫色标签 | 浅紫背景 + 深紫文字 |
| `++文本++` | 绿色标签 | 浅绿背景 + 深绿文字 |
| `!!文本!!` | 高亮标签 | 红色背景 + 白色文字 |

## 📤 导入与导出

### 导入 Markdown

点击 **「📂 导入」** 按钮，选择本地的 `.md` 或 `.txt` 文件，编辑器内容会被替换为文件内容。

### 导出 Markdown

点击 **「💾 导出」** 按钮，将当前简历内容下载为 `.md` 文件，文件名自动包含日期（如 `简历_2026-2-14.md`）。

### 导出 PDF

点击右上角 **「导出 PDF」** 按钮，弹出下拉菜单提供两种方案：

| 方案 | 说明 | 推荐度 |
|------|------|--------|
| **打印 / 保存 PDF** | 使用浏览器原生打印功能，在打印对话框中选择「另存为 PDF」 | ⭐ 推荐 |
| **直接导出 PDF** | 使用 html2canvas 截图转 PDF，无需打印对话框 | 备用 |

> **💡 推荐使用「打印 / 保存 PDF」**：
> - 该方案样式还原度最高，支持智能分页
> - 在打印对话框中确保选择：纸张大小 **A4**、边距 **无**、勾选 **背景图形**
> - 目标打印机选择 **另存为 PDF / Microsoft Print to PDF**

### 重置模板

点击 **「🔄 重置」** 按钮，可将编辑器内容恢复为内置的默认简历模板（需确认）。

## 📏 行间距调节

顶部导航栏中的 **「行距」滑块** 可实时调整简历正文的行间距：

- 范围：**18px ~ 32px**
- 默认值：**22px**
- 调整后自动保存，下次打开保持上次设置

---

## 💾 自动保存

所有编辑内容和行间距设置会**自动保存**到浏览器的 localStorage：

- 内容保存有 **500ms 防抖**，减少频繁写入
- 刷新页面或关闭浏览器后再打开，内容不会丢失
- 清除浏览器数据会导致保存内容丢失，建议定期使用「导出」功能备份
